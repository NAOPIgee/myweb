# myweb 個人簡歷網站

## 使用技術

### 前端
* **Framework**: Next.js 14+ (App Router)
* **Language**: TypeScript
* **Styling**: Tailwind CSS + CSS Modules (無 Bootstrap)
* **HTTP Client**: Axios
* **Design**: 響應式復古報紙風格

### 後端
* **Framework**: ASP.NET Core Web API (.NET 8)
* **Database**: MySQL (透過 Pomelo.EntityFrameworkCore.MySql)
* **ORM**: Entity Framework Core
* **Auth**: BCrypt (密碼雜湊)
